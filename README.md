\## Scientific Question



Does object weight affect movement smoothness in chronic stroke patients of reach-to-grasp movements, as measured by Number of Velocity Peaks (NVP) and Log Dimensionless Jerk (LDLJ)?



\## Dataset



\*\*Source:\*\* Zenodo 3930752 - Kinematics of reach-to-grasp after stroke  

\*\*Subjects:\*\* 10 chronic stroke patients  

\*\*Tasks:\*\* Reach-to-grasp with 3 block weights (108g, 490g, 1008g)



\## Repository Structure



\- `main.ipynb` — Python analysis

\- `main.Rmd` — R analysis

\- `main.Rproj` — RStudio project

\- `data/` — Raw IMU data

\- `results/` — Analysis outputs (CSV + figures)

\- `sources/` — Custom Python functions

\- `LICENCE` — MIT License



\## Key Results



\- \*\*7,740 movements analyzed\*\*

\- \*\*NVP range:\*\* 1.0 - 9.0

\- \*\*LDLJ range:\*\* -13.0 to -11.0

\- \*\*Finding:\*\* Block weight significantly affects movement smoothness
