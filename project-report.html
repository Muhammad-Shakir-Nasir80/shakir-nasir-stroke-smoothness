Introduction

Stroke is a leading cause of disability worldwide, affecting motor control and movement quality in millions of patients annually. One of the most impactful functional limitations following stroke is the loss of dexterity and smoothness in upper limb movements, particularly during fine motor tasks such as reach-to-grasp movements. Understanding how task parameters—such as object weight—affect movement quality is crucial for designing effective rehabilitation protocols and predicting functional recovery.

Movement smoothness is a sensitive indicator of motor control integrity and neuromuscular efficiency. Smooth, coordinated movements reflect efficient motor planning and execution, whereas fragmented or jerky movements suggest decomposition of movement control. In healthy individuals, reach-to-grasp movements typically exhibit a single, bell-shaped velocity profile. In contrast, stroke patients often demonstrate multi-peaked velocity profiles indicative of impaired motor control and increased compensatory strategies.

While previous literature has examined movement quality in stroke, few studies have systematically investigated how object weight—a common variable in activities of daily living—modulates smoothness metrics in chronic stroke survivors. This gap motivated the current investigation.

Scientific Question

Does object weight affect movement smoothness in chronic stroke patients during reach-to-grasp movements, as measured by Number of Velocity Peaks (NVP) and Log Dimensionless Jerk (LDLJ)?

Hypothesis

We hypothesize that heavier objects will produce less smooth movements compared to lighter objects, because increased load increases the neuromuscular demand on an already impaired motor system. Additionally, we expect both NVP and LDLJ to show differential responses to weight changes, allowing cross-validation of smoothness degradation.

Methods

2.1 Dataset

Source: Schwarz et al. (2020). Kinematics of reach-to-grasp and displacement after stroke. Dataset published on Zenodo (DOI: 10.5281/zenodo.3930752).

Parameter Value
Subjects 10 chronic stroke patients
Block weights ARAT_BL (108g - Light), ARAT_BW (490g - Medium), ARAT_BH (1008g - Heavy)
Total movements analyzed 7,740 individual reach-to-grasp movements
Sensors 8 IMU units (3-axis accelerometer + 3-axis gyroscope)
Sampling frequency 100 Hz (accelerometer), 200 Hz (gyroscope)
File format .mat (MATLAB binary format)

2.2 Analysis Pipeline

Phase 1: Python Analysis (main.ipynb)

  1. Data loading: Extracted upper arm acceleration from .mat files using scipy.io.loadmat()
  2. Signal processing: Applied Butterworth low-pass filter (10 Hz cutoff, 4th order, zero-phase using filtfilt) to remove high-frequency noise while preserving movement features
  3. Movement segmentation: Detected individual movement epochs using acceleration magnitude thresholding (15th percentile), with minimum duration of 300 ms to exclude noise artifacts
  4. Velocity computation: Integrated filtered acceleration separately for each movement epoch to eliminate cumulative drift and ensure epoch-specific velocity profiles
  5. Metric calculation: Computed NVP and LDLJ for each movement epoch
  6. Output: Generated smoothness_results.csv with 7,740 individual movements

Phase 2: R Analysis (main.Rmd)

  1. Data import: Loaded CSV from Python using read.csv()
  2. Normality assessment: Applied Shapiro-Wilk test and generated QQ plots
  3. Statistical testing: Performed Wilcoxon rank-sum test (Light vs Heavy blocks) due to non-normal distributions
  4. Visualization: Created publication-quality box plots using ggplot2 with color-coded block weights
  5. Interpretation: Computed effect sizes and descriptive statistics

2.3 Movement Smoothness Metrics

Metric Definition/Interpretation Clinical Meaning
NVP
(Number of Velocity Peaks)		 Count of local maxima in velocity profile above threshold (1% of peak velocity) Higher = more fragmented; 1 = single smooth peak; >1 = decomposition
LDLJ
(Log Dimensionless Jerk)		 −log(T⁵/D² × ∫jerk²dt), normalized by duration and displacement More negative = less smooth; dimensionless allows cross-dataset comparison

Why these metrics? NVP provides intuitive interpretation of movement decomposition. LDLJ offers dimensionless, normalized quantification of jerk energy with published high reliability (ICC > 0.9) in stroke literature. Together, they provide complementary perspectives on smoothness.

3.5 Key Findings Summary

Primary Results:
  • Optimal weight for smoothness: Medium block (490g) produced smoothest movements with lowest NVP median (2.0)
  • Light block effects: 108g block resulted in higher fragmentation (NVP median = 3.0) and less smooth jerk profiles
  • Heavy block effects: 1008g block showed similar NVP (median = 3.0) to light block, suggesting U-shaped relationship
  • Statistical significance: Wilcoxon test p = 0.023-0.042, confirming weight effects are not due to chance
  • Non-linear relationship: Medium weight appears optimal, not following monotonic degradation pattern

Discussion

4.1 Interpretation of Results

Our results reveal a nuanced, non-linear relationship between object weight and movement smoothness in chronic stroke patients. The finding that medium weight (490g) produces the smoothest movements—rather than lighter objects producing the smoothest movements—suggests that proprioceptive feedback plays a crucial role in stroke motor control.

Proposed mechanisms:

  1. Proprioceptive threshold hypothesis: Very light objects (108g) may not provide sufficient proprioceptive feedback for optimal motor planning, leading to movement decomposition and higher NVP
  2. Optimal load window: Medium weight (490g) appears to represent a "sweet spot" where proprioceptive signals are robust enough to guide smooth execution without exceeding remaining motor capacity
  3. Saturation at heavy loads: While heavier objects (1008g) provide stronger proprioceptive cues, they also challenge motor output capacity, resulting in similar fragmentation levels as light objects

This non-monotonic relationship contradicts the simple prediction that heavier loads always degrade performance. Instead, it suggests a physiologically meaningful optimal load range for chronic stroke motor control.

4.2 Methodological Strengths

  • Large movement dataset: 7,740 individual movements analyzed, providing robust statistical power despite small subject sample
  • Dual complementary metrics: NVP and LDLJ provide independent perspectives on smoothness, with convergent findings strengthening conclusions
  • Rigorous signal processing: Zero-phase filtering eliminates temporal distortion; epoch-based integration prevents drift artifacts
  • Transparent analysis pipeline: Python for computation, R for validation and publication-quality visualization
  • Open science: All code, results, and analysis publicly available on GitHub for full reproducibility and community verification

4.3 Study Limitations

  • Small subject sample: N=10 subjects limits generalizability; results should be confirmed in larger, multi-site cohorts (target N ≥ 30)
  • Absence of healthy control group: Inability to quantify absolute impairment severity or benchmark smoothness values against age-matched healthy individuals
  • Cross-sectional design: Cannot assess longitudinal recovery trajectories, training effects, or stability of findings over time
  • Single measurement occasion: Unknown whether within-subject variability or practice effects influenced results
  • Limited spatial analysis: Original hypothesis regarding upper vs lower hemispace not tested in this dataset
  • No kinematic details: Analysis focused on smoothness metrics; underlying joint angles, muscle activation patterns not examined

4.4 Clinical and Rehabilitative Implications

These findings have important implications for rehabilitation design and motor re-learning protocols:

  • Task-specific load optimization: Rehabilitation programs should consider intermediate load ranges (400-600g) as potentially optimal for promoting smooth movement patterns
  • Proprioceptive feedback: Results suggest that adequate proprioceptive cuing is essential for motor control; excessively light objects should be avoided
  • Individualization: Given heterogeneity in motor impairment, optimal loads should be determined individually rather than applying fixed prescriptions
  • Outcome measurement: NVP and LDLJ show promise as sensitive markers of motor recovery in longitudinal intervention studies
---

Recommendations for Future Research

  1. Larger sample sizes: Conduct studies with N ≥ 30-50 subjects to improve statistical power and generalizability across stroke populations
  2. Healthy control group: Include age- and sex-matched healthy controls to establish normative smoothness ranges and quantify absolute impairment
  3. Longitudinal designs: Follow subjects over 6-12 months to assess whether optimal load changes during recovery and to model recovery trajectories
  4. Hemispace analysis: Test original hypothesis regarding upper vs lower reaching targets and their interaction with load effects
  5. Limb comparison: Compare affected vs unaffected limb performance to examine asymmetries and potential bilateral transfer
  6. Dose-response characterization: Test finer weight gradations (e.g., 100g, 200g, 300g, ..., 900g) to map smoothness response curves
  7. Mechanistic studies: Use musculoskeletal modeling and EMG to link smoothness metrics to underlying muscle activation patterns and motor strategy changes
  8. Multi-metric validation: Incorporate additional smoothness metrics (SPARC, normalized jerk, acceleration smoothness) for cross-validation
---

Conclusion

This study investigated the effect of object weight on movement smoothness in 10 chronic stroke patients during reach-to-grasp movements, analyzing 7,740 individual movement episodes using two complementary metrics—Number of Velocity Peaks (NVP) and Log Dimensionless Jerk (LDLJ).

Primary finding: Object weight significantly affects movement smoothness (p < 0.05), with a non-linear relationship where medium weight (490g) produced the smoothest movements. This challenges the simple prediction that heavier loads monotonically degrade performance and suggests instead an optimal load range where proprioceptive feedback enhances motor control without overwhelming residual motor capacity.

While our subject sample is modest and requires confirmation in larger cohorts, this work demonstrates the feasibility and value of high-throughput kinematic analysis combining Python for computation and R for statistical validation. The publicly available code and data repository on GitHub establish a template for reproducible biomechanical research accessible to the broader scientific community.

These results have direct relevance for rehabilitation design, suggesting that task-specific object weights should be carefully selected to optimize proprioceptive feedback. Future work incorporating healthy controls, larger sample sizes, and longitudinal designs will clarify the mechanisms by which object weight modulates movement smoothness and its relationship to functional recovery trajectories in stroke populations.

Appendix: Technical Implementation Details

A.1 Software Environment

  • Python 3.10: Data loading (scipy.io), signal processing (scipy.signal), numerical computation (numpy, pandas)
  • R 4.2: Statistical analysis (rstatix), data visualization (ggplot2, gridExtra), descriptive statistics
  • Git/GitHub: Version control with 5 feature branches (main, develop, feature/python-analysis, feature/r-analysis, feature/documentation)

A.2 Signal Processing Parameters

  • Butterworth filter: 10 Hz cutoff frequency, 4th order, zero-phase (filtfilt implementation)
  • Movement detection: Acceleration magnitude thresholding at 15th percentile
  • Minimum movement duration: 300 ms to exclude noise and micro-movements
  • NVP threshold: 1% of maximum velocity in epoch
  • LDLJ normalization: T⁵/D² normalization yielding dimensionless metric

A.3 Statistical Methods

  • Normality assessment: Shapiro-Wilk test (α = 0.05) with QQ plot visualization
  • Group comparison: Wilcoxon rank-sum test (two-sided, α = 0.05)
  • Effect size: Rank biserial correlation r = 1 − (2W)/(n₁·n₂)
  • Descriptive statistics: Median, interquartile range, mean, standard deviation
